# Bristol-Pink Project Notes

This repository contains several early test versions of the application source code.

For the formal production-ready version, please check the files inside the `run` folder.

You can run `run_bristol_pink_new.bat` to compile the project, configure the environment, and launch the web application in one click.

## Repository Structure

- `frontend_app.py` - extracted frontend/UI layer for the Streamlit web app
- `backend_core.py` - data loading, preprocessing, forecasting, and evaluation logic
- `run/` - formal version files

## Important Note

The code files in the repository root are initial testing iterations. The official version should be reviewed from the `run` directory.
